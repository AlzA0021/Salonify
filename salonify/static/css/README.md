# Salonify CSS Final

این بستهٔ CSS بازسازی و بهینه‌سازی شده‌ی کامل برای مسیر `salonify/static/css` است.

## فایل‌ها
- `base.css` — ریست + utilities + تمام `@font-face`های یکتا
- `tokens.css` — متغیرهای طراحی (رنگ/تایپو/فاصله/Radius)
- `common.css` — قوانین مشترک استخراج‌شده بین فایل‌ها (با نگه‌داری aliases برای سازگاری)
- پوشهٔ ساختار اصلی پروژه (pages/ و …) — با قوانین ویژه‌ی هر صفحه، بعد از حذف موارد تکراری

**Import پیشنهادی در هر صفحه:**
```css
@import "./base.css";
@import "./common.css";
/* سپس CSS صفحهٔ خودتان (در صورت نیاز tokens.css را قبل از base.css ایمپورت کنید اگر جدا نگه می‌دارید) */
```

## تغییرنام کلاس‌ها (اختیاری اما توصیه‌شده)
فایل `final_rename_map.csv` فهرست کلاس‌های هم‌معنی را به یک نام مرجع پیشنهاد می‌کند.
در `common.css` همهٔ aliasها کنار هم گروه‌بندی شده‌اند تا بدون تغییر HTML کار کند.
برای تمیزکاری نهایی، در قالب‌ها کلاس‌های قدیمی را به جدید تغییر دهید.
